# 宝塔面板部署教程 - 视频下载工具

## 一、环境准备

### 1.1 服务器要求
- 操作系统：CentOS 7+ / Ubuntu 18+ / Debian 9+
- 内存：至少 2GB RAM
- 硬盘：至少 20GB 可用空间
- 域名：downloadhd.net（已解析到服务器 IP）

### 1.2 安装宝塔面板

```bash
# CentOS 安装命令
yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh

# Ubuntu/Debian 安装命令
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh
```

安装完成后记录：
- 宝塔面板地址
- 用户名
- 密码

## 二、宝塔面板配置

### 2.1 登录宝塔面板
访问 `http://你的服务器IP:8888`，使用安装时的账号密码登录。

### 2.2 安装软件环境

在宝塔面板 → 软件商店，安装以下软件：

**前端环境：**
- Nginx 1.22+
- Node.js 18+ (使用 PM2 管理)

**后端环境：**
- Python 3.10+
- Redis 7.0+

**其他：**
- SSL 证书管理器

## 三、项目部署

### 3.1 上传项目文件

1. 在宝塔面板 → 文件，创建目录：
   ```
   /www/wwwroot/downloadhd.net
   ```

2. 上传项目文件到该目录，或使用 Git：
   ```bash
   cd /www/wwwroot/downloadhd.net
   git clone 你的仓库地址 .
   ```

### 3.2 前端部署

```bash
cd /www/wwwroot/downloadhd.net

# 安装依赖
npm install

# 构建生产版本
npm run build
```

构建完成后，`dist` 目录包含所有前端静态文件。

### 3.3 后端部署

```bash
cd /www/wwwroot/downloadhd.net/backend

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
nano .env  # 编辑配置
```

编辑 `.env` 文件：
```env
REDIS_HOST=127.0.0.1
REDIS_PORT=6379
REDIS_DB=0
API_BASE_URL=https://downloadhd.net
```

### 3.4 使用 PM2 管理后端进程

在宝塔面板 → Node.js 版本管理 → PM2 管理器：

1. 点击"添加项目"
2. 配置如下：
   - 项目名称：`downloadhd-backend`
   - 启动文件：`/www/wwwroot/downloadhd.net/backend/main.py`
   - 运行目录：`/www/wwwroot/downloadhd.net/backend`
   - 启动命令：`python3 main.py`
   - 端口：`8000`

或使用命令行：
```bash
cd /www/wwwroot/downloadhd.net/backend
pm2 start main.py --name downloadhd-backend --interpreter python3
pm2 save
pm2 startup
```

## 四、Nginx 配置

### 4.1 创建网站

1. 宝塔面板 → 网站 → 添加站点
   - 域名：`downloadhd.net`
   - 根目录：`/www/wwwroot/downloadhd.net/dist`
   - PHP 版本：纯静态

### 4.2 配置 Nginx 反向代理

点击网站 → 设置 → 配置文件，修改为：

```nginx
server {
    listen 80;
    listen 443 ssl http2;
    server_name downloadhd.net www.downloadhd.net;

    # SSL 证书配置（稍后配置）
    ssl_certificate /www/server/panel/vhost/cert/downloadhd.net/fullchain.pem;
    ssl_certificate_key /www/server/panel/vhost/cert/downloadhd.net/privkey.pem;

    # 强制 HTTPS
    if ($server_port !~ 443){
        rewrite ^(/.*)$ https://$host$1 permanent;
    }

    # 前端静态文件
    root /www/wwwroot/downloadhd.net/dist;
    index index.html;

    # Gzip 压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml application/xml application/xml+rss text/javascript;

    # 静态文件缓存
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|svg|woff|woff2|ttf|eot)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }

    # API 反向代理
    location /api/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_connect_timeout 300s;
        proxy_send_timeout 300s;
        proxy_read_timeout 300s;
    }

    # 前端路由支持（SPA）
    location / {
        try_files $uri $uri/ /index.html;
    }

    # robots.txt 和 sitemap.xml
    location = /robots.txt {
        root /www/wwwroot/downloadhd.net/dist;
    }

    location = /sitemap.xml {
        root /www/wwwroot/downloadhd.net/dist;
    }

    # 安全头
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;

    access_log /www/wwwroot/downloadhd.net/logs/access.log;
    error_log /www/wwwroot/downloadhd.net/logs/error.log;
}
```

### 4.3 配置 SSL 证书（HTTPS）

1. 宝塔面板 → 网站 → 设置 → SSL
2. 选择"Let's Encrypt"免费证书
3. 勾选域名：`downloadhd.net` 和 `www.downloadhd.net`
4. 点击"申请"
5. 开启"强制 HTTPS"

## 五、更新 sitemap.xml 和 robots.txt

### 5.1 更新 sitemap.xml

编辑 `/www/wwwroot/downloadhd.net/public/sitemap.xml`：

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://downloadhd.net/</loc>
    <lastmod>2026-03-12</lastmod>
    <changefreq>daily</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://downloadhd.net/contact</loc>
    <lastmod>2026-03-12</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
  <url>
    <loc>https://downloadhd.net/terms</loc>
    <lastmod>2026-03-12</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.6</priority>
  </url>
  <url>
    <loc>https://downloadhd.net/tutorial-ios</loc>
    <lastmod>2026-03-12</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.7</priority>
  </url>
</urlset>
```

### 5.2 更新 robots.txt

编辑 `/www/wwwroot/downloadhd.net/public/robots.txt`：

```
User-agent: *
Allow: /
Disallow: /api/

Sitemap: https://downloadhd.net/sitemap.xml
```

### 5.3 重新构建前端

```bash
cd /www/wwwroot/downloadhd.net
npm run build
```

## 六、谷歌收录配置

### 6.1 验证网站所有权

1. 访问 [Google Search Console](https://search.google.com/search-console)
2. 点击"添加资源" → "网址前缀"
3. 输入：`https://downloadhd.net`
4. 选择验证方式：

**方式一：HTML 文件验证**
- 下载验证文件（如 `google1234567890abcdef.html`）
- 上传到 `/www/wwwroot/downloadhd.net/dist/` 目录
- 点击"验证"

**方式二：HTML 标签验证**
- 复制 meta 标签
- 添加到 `index.html` 的 `<head>` 部分：
  ```html
  <meta name="google-site-verification" content="你的验证码" />
  ```
- 重新构建：`npm run build`
- 点击"验证"

### 6.2 提交 Sitemap

1. 在 Google Search Console 左侧菜单 → "站点地图"
2. 输入：`https://downloadhd.net/sitemap.xml`
3. 点击"提交"

### 6.3 请求编入索引

1. 在 Google Search Console 顶部搜索框输入你的网址
2. 点击"请求编入索引"
3. 等待谷歌抓取（通常 1-7 天）

### 6.4 配置 Google Analytics（可选）

1. 访问 [Google Analytics](https://analytics.google.com/)
2. 创建账号和媒体资源
3. 获取跟踪 ID（如 `G-XXXXXXXXXX`）
4. 在 `index.html` 的 `<head>` 添加：

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```

## 九、常见问题

### 9.1 网站无法访问
- 检查防火墙：宝塔面板 → 安全，放行 80 和 443 端口
- 检查域名解析是否生效：`ping downloadhd.net`
- 检查 Nginx 是否运行：`systemctl status nginx`

### 9.2 API 请求失败
- 检查后端是否运行：`pm2 status`
- 查看后端日志：`pm2 logs downloadhd-backend`
- 检查 Redis 是否运行：`systemctl status redis`

### 9.3 SSL 证书过期
- Let's Encrypt 证书有效期 90 天
- 宝塔面板会自动续期
- 手动续期：宝塔面板 → 网站 → SSL → 续签

### 9.4 谷歌未收录
- 确认 robots.txt 没有禁止抓取
- 检查 sitemap.xml 格式是否正确
- 在 Google Search Console 查看抓取错误
- 确保网站内容原创且有价值
- 等待时间：新站通常需要 1-4 周
